# 🚀 SauceDemo Test Automation (Cypress + Bug Reporting)

Este repositorio contiene la automatización de pruebas end-to-end para la aplicación **SauceDemo**, utilizando Cypress, junto con el reporte de bugs Trello.

El objetivo del proyecto es automatizar los módulos principales de la aplicación y documentar defectos encontrados durante la ejecución de pruebas.

## 📌 Objetivo del proyecto

- Automatizar casos de prueba de SauceDemo utilizando Cypress.
- Aplicar buenas prácticas de testing automatizado.
- Reportar bugs encontrados en la aplicación.
- Documentar evidencias en Trello.

## 🧪 Casos automatizados

### Login
- Login exitoso con usuario válido
- Login con usuario bloqueado
- Validación de errores de login

### Inventario
- Visualización de productos
- Ordenamiento de productos por precio
- Validación de elementos del inventario

### Carrito
- Agregar productos al carrito
- Validar cantidad de productos en el carrito
- Eliminar productos del carrito

### Checkout
- Completar formulario de checkout
- Validaciones de campos obligatorios
- Flujo completo de compra

## 🐞 Reporte de bugs

Los bugs encontrados durante la automatización fueron documentados en Trello.
